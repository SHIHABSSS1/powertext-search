How to Run the Data Search Website
================================

Prerequisites:
-------------
1. Python 3.7 or higher installed on your computer
2. The data.xlsx file in the same directory as the application files

Steps to Run:
------------
1. Open Command Prompt (Windows) or Terminal (Mac/Linux)

2. Navigate to the project directory:
   cd C:\Web_Development\project

3. Install the required packages (if not already installed):
   pip install -r requirements.txt

4. Run the Flask application:
   python app.py

5. Open your web browser and go to:
   http://localhost:5000

The website should now be running and you can:
- Enter search keywords in the search box
- Press Enter or click the Search button
- View the results displayed with spaces between data values

To stop the website:
- Press Ctrl+C in the command prompt/terminal where the application is running

Note: Make sure the data.xlsx file remains in the same directory as app.py
